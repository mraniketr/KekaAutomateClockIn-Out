# KekaAutomateClockIn-Out
Using puppeter to automate daily task of clockin/out in Keka Attendance portal


Create a .env file as following

EMAIL=<your org email>
PASSWORD=<respective pwd>
URI=https://app.keka.com/Account/Login?ReturnUrl=%2F
SUBDOMAINURI=<organisation keka subdomain url after login page>
LATITUDE=<coordinates>
LONGITUDE=<coordinates>
HEADLESS=<true/false>
